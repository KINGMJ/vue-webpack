# vue-webpack
> 使用 vue-cli 构建的 vue-demo，包含官方基础教程和示例
 
## 安装及运行

```
npm install
```

### 运行开发环境

```
npm run serve
```

### 运行生产环境

```
npm run build
```

### 文件检查

```
npm run lint
```
